# Reto 1: Calculadora de Sumas

Programa que lee datos de la entrada estándar, limpia caracteres no numéricos, 
trunca los valores a enteros y entrega la suma total.

## Ejecución
Para ejecutar el programa con un archivo de prueba:
`python main.py < datos.txt`

## Características
- Soporta números positivos y negativos.
- Ignora caracteres no numéricos automáticamente.
- Realiza truncado de decimales (no redondeo).
- Maneja líneas vacías o con espacios.
